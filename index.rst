####################################################
Dydra Application Programming Interface (API) Manual
####################################################

This is the application programming interface reference manual for `Dydra`_,
the cloud-hosted RDF & SPARQL database service.

The latest version of this manual can be browsed online at `api.dydra.com`_,
and it is also available for download in the `PDF`_ and `EPUB`_ formats.

.. _Dydra:         http://dydra.com/
.. _api.dydra.com: http://api.dydra.com/
.. _PDF:           http://api.dydra.com/download/dydra-api.pdf
.. _EPUB:          http://api.dydra.com/download/dydra-api.epub
